# 每日签到功能说明

## 📋 功能概述

基于 Redis + MySQL 实现的每日签到系统，支持：
- ✅ 每日签到一次（0 点刷新）
- ✅ 连续签到天数统计
- ✅ 断签重置计数器
- ✅ 特殊奖励（第 7 天、第 30 天）
- ✅ 积分和虚拟货币双重奖励

## 🎁 奖励规则

### 基础奖励
- **每日签到**：+10 积分 + 1 元虚拟货币

### 特殊奖励
- **连续签到 7 天**：额外 +88 积分 + 10 元
- **连续签到 30 天**：额外 +888 积分 + 100 元 + 手机兑换券

## 🗄️ 数据存储方案

### Redis（缓存层）
- **Key 格式**：`check_in:{userId}:{date}`
  - 存储今日是否已签到
  - 过期时间：30 天
- **连续签到 Key**：`consecutive_days:{userId}`
  - 存储当前连续签到天数
  - 过期时间：30 天

### MySQL（持久化）
```sql
daily_check_in_records 表：
- id: 主键
- user_id: 用户 ID
- check_in_date: 签到日期
- reward_points: 奖励积分
- reward_balance: 奖励虚拟货币
- consecutive_days: 连续签到天数
- created_at: 创建时间
```

在 `game-service/src/main/resources/application.yml` 中配置 Redis：
```yaml
spring:
  redis:
    host: localhost
    port: 6379
    password: 
    database: 0
    timeout: 5000ms
```

### 4. 添加 Maven 依赖
已在 `game-service/pom.xml` 中添加：
```xml

    <artifactId>spring-boot-starter-data-redis</artifactId>
```

### 5. 启动服务
```bash
cd game-service
mvn clean package
java -jar target/game-service-1.0.0-SNAPSHOT.jar
```

## ⚙️ 核心逻辑说明

### 1. 连续签到计算
```java
// 查询上次签到日期
LocalDate lastDate = getLastCheckInDate(userId);
LocalDate today = LocalDate.now();
LocalDate yesterday = today.minusDays(1);

if (lastDate.equals(yesterday) || lastDate.equals(today)) {
    // 继续累加
    return lastRecord.getConsecutiveDays() + 1;
} else {
    // 断签重置
    return 1;
}
```

### 2. 防重复签到
```java
// Redis 检查
String checkInKey = "check_in:" + userId + ":" + today;
Boolean hasCheckedIn = redisTemplate.hasKey(checkInKey);
if (Boolean.TRUE.equals(hasCheckedIn)) {
    return "今日已签到";
}
```

## 🔍 常见问题

### Q1: Redis 挂了怎么办？
A: 系统会降级到只使用数据库，但性能会下降。建议配置 Redis 哨兵或集群。

### Q2: 跨天如何处理？
A: 每次签到时动态计算日期，Redis Key 包含日期信息，自动在 0 点刷新。

### Q3: 如何查看用户签到历史？
A: 直接查询数据库表 `daily_check_in_records`。

### Q4: 如何修改奖励数值？
A: 在 `CheckInServiceImpl.java` 中修改常量定义。

## 📊 监控与统计

### 推荐监控指标
- 每日签到人数
- 平均连续签到天数
- 特殊奖励发放次数
- Redis 命中率

### SQL 统计示例
```sql
-- 今日签到人数
SELECT COUNT(DISTINCT user_id) 
FROM daily_check_in_records 
WHERE check_in_date = CURDATE();

-- 连续签到 7 天以上的人数
SELECT COUNT(*) FROM (
    SELECT user_id, MAX(consecutive_days) as max_days
    FROM daily_check_in_records
    GROUP BY user_id
    HAVING max_days >= 7
) t;
```
