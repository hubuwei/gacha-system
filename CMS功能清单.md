## 🔧 技术实现

### 前端技术栈
- React 18
- Ant Design 5.x（UI组件库）
- @ant-design/charts（图表库）
- React Router 6（路由管理）
- Axios（HTTP请求）

### 后端技术栈
- Spring Boot 2.7.x
- Spring Data JPA
- MySQL 8.0
- Redis（缓存）

### API接口设计

#### Dashboard相关
```
GET  /api/cms/dashboard/stats           # 获取统计数据
GET  /api/cms/dashboard/weekly-revenue  # 获取本周营收
GET  /api/cms/dashboard/popular-games   # 获取热门游戏排行
```

#### 评论审核相关
```
GET    /api/cms/reviews                 # 获取评论列表
PUT    /api/cms/reviews/{id}/approve    # 审核通过
PUT    /api/cms/reviews/{id}/reject     # 审核拒绝
DELETE /api/cms/reviews/{id}            # 删除评论
```

#### Banner配置相关
```
GET    /api/cms/banners                 # 获取Banner列表
POST   /api/cms/banners                 # 新增Banner
PUT    /api/cms/banners/{id}            # 更新Banner
DELETE /api/cms/banners/{id}            # 删除Banner
PUT    /api/cms/banners/{id}/status     # 切换状态
```

## 🚀 快速开始

### 本地开发
```bash
# 启动前端
cd cms-admin
npm install
npm run dev

# 启动后端
cd cms-service
mvn spring-boot:run
```

### 访问地址
- CMS后台：http://localhost:5174/cms/login
- 默认管理员账号：admin / admin123

## 💡 使用建议

1. **每日必查**：登录CMS后首先查看首页数据看板，了解今日业务情况

3. **热门分析**：通过热门游戏排行调整营销策略和库存
4. **内容审核**：定期审核新评论，维护社区环境
