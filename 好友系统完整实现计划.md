**需求**：

**技术方案**：
```
方案A：WebSocket（推荐）
- 优点：实时性最好，Steam同款体验
- 缺点：需要维护连接，服务器压力较大

方案B：SSE (Server-Sent Events)
- 优点：单向推送，实现简单
- 缺点：只能服务端→客户端

方案C：轮询（最简单）
- 优点：实现最简单
- 缺点：实时性差，浪费资源
```

**实现步骤**：
1. 添加WebSocket依赖
2. 创建WebSocket配置类
3. 创建NotificationService
4. 前端建立WebSocket连接

#### 1.2 站内信系统
**需求**：
- 存储所有通知记录
- 支持已读/未读标记
- 支持通知类型分类

**数据库表**：
```sql
CREATE TABLE user_notification (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    uid BIGINT NOT NULL,              -- 接收人ID
    type VARCHAR(50) NOT NULL,        -- 通知类型
    title VARCHAR(200),               -- 标题
    content TEXT,                     -- 内容
    is_read TINYINT DEFAULT 0,        -- 是否已读
    related_id BIGINT,                -- 关联ID（如申请ID）
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    INDEX idx_uid (uid),
    INDEX idx_is_read (is_read)
);
```

## 💡 技术要点

### 1. WebSocket实现示例

**后端配置**：
```java
@Configuration
@EnableWebSocketMessageBroker
public class WebSocketConfig implements WebSocketMessageBrokerConfigurer {
    @Override
    public void configureMessageBroker(MessageBrokerRegistry config) {
        config.enableSimpleBroker("/topic");
        config.setApplicationDestinationPrefixes("/app");
    }
}
```

**推送通知**：
```java
@Service
public class NotificationService {
    @Autowired
    private SimpMessagingTemplate messagingTemplate;
    
    public void sendFriendRequestNotification(Long receiveUid, Long applyUid) {
        Map<String, Object> notification = new HashMap<>();
        notification.put("type", "FRIEND_REQUEST");
        notification.put("from", applyUid);
        notification.put("message", "有人想加你为好友");
        
        messagingTemplate.convertAndSendToUser(
            receiveUid.toString(),
            "/topic/notifications",
            notification
        );
    }
}
```

**前端连接**：
```javascript
import SockJS from 'sockjs-client';
import Stomp from 'stompjs';

const socket = new SockJS('http://localhost:8081/ws');
const stompClient = Stomp.over(socket);

stompClient.connect({}, frame => {
    stompClient.subscribe('/user/topic/notifications', message => {
        const notification = JSON.parse(message.body);
        showNotification(notification);
    });
});
```
