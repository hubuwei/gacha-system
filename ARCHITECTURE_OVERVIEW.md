## 🔄 数据流向图

### 用户登录流程
```
用户 → Frontend → Auth Service → MySQL (验证)
                              ↓
                         生成 JWT Token
                              ↓
用户 ← Frontend ← Auth Service (返回 Token)
```

### 游戏购买流程
```
用户 → Frontend → Mall Service → Auth Service (验证 Token)
                              ↓
                         Mall Service → MySQL (创建订单)
                              ↓
                         Mall Service → Wallet Service (扣款)
                              ↓
                         Mall Service → RabbitMQ (异步通知)
                              ↓
用户 ← Frontend ← Mall Service (返回结果)
```

### 抽卡流程
```
用户 → Frontend → Gacha Service → Auth Service (验证)
                              ↓
                         Gacha Service → MySQL (检查余额)
                              ↓
                         Gacha Service → 随机算法 (计算结果)
                              ↓
                         Gacha Service → MySQL (记录抽取)
                              ↓
                         Gacha Service → RabbitMQ (发放奖励)
                              ↓
用户 ← Frontend ← Gacha Service (返回结果)
```

## 🎯 关键技术选型理由

| 技术 | 选择理由 | 替代方案 |
|------|---------|---------|
| Spring Boot | 成熟稳定，生态完善 | Quarkus, Micronaut |
| MySQL | 关系型数据首选 | PostgreSQL, MariaDB |
| Redis | 高性能缓存 | Memcached |
| Elasticsearch | 强大的搜索能力 | Solr, Meilisearch |
| RabbitMQ | 可靠的消息队列 | Kafka, RocketMQ |
| React | 组件化，生态丰富 | Vue, Angular |
| Nginx | 高性能反向代理 | Apache, Traefik |
| JWT | 无状态认证 | Session, OAuth2 |
