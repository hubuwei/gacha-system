## 🔧 核心配置说明

### Redis 配置（application.yml）
```yaml
spring:
  redis:
    host: localhost
    port: 6379
    password: 
    database: 0
    timeout: 5000ms
    lettuce:
      pool:
        max-active: 8
        max-wait: -1ms
        max-idle: 8
        min-idle: 0
```

### Maven 依赖（pom.xml）
```xml
<!-- Redis -->
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-data-redis</artifactId>
</dependency>
<!-- Redis 连接池 -->
<dependency>
    <groupId>org.apache.commons</groupId>
    <artifactId>commons-pool2</artifactId>
</dependency>
```

## 🎯 功能特性

### ✅ 已实现的功能
1. **每日签到**：每个用户每天只能签到一次
2. **刷新机制**：每天 0 点自动刷新
3. **连续签到统计**：记录并累加连续签到天数
4. **断签重置**：中断签到后计数器归零
5. **特殊奖励**：
   - 第 7 天：额外 +88 积分 +10 元
   - 第 30 天：额外 +888 积分 +100 元 + 手机兑换券
6. **双重奖励**：积分 + 虚拟货币
7. **防重复签到**：Redis + 数据库双重校验

### 📊 数据存储
- **Redis**：临时存储今日签到状态和连续天数（30 天过期）
- **MySQL**：永久存储签到历史记录

## 🚀 启动顺序

1. **启动 Redis**
   ```bash
   redis-server.exe redis.conf
   ```

2. **创建数据库表**
   ```bash
   mysql -u root -p gacha_system < database/create_checkin_table.sql
   ```

3. **编译项目**
   ```bash
   cd game-service
   mvn clean package
   ```

4. **启动服务**
   ```bash
   java -jar target/game-service-1.0.0-SNAPSHOT.jar
   ```

5. **测试功能**
   - 打开 `checkin.html`
   - 或使用 Postman/cURL 测试 API

## ⚠️ 注意事项

1. **Redis 必须启动**：否则服务无法启动
2. **数据库表必须先创建**：JPA 不会自动创建带索引的表
3. **端口不能冲突**：Redis(6379)、Game-Service(8082)
4. **时区设置**：确保服务器时区正确（中国为 UTC+8）
