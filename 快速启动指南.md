## 🔥 快速启动命令

### 第1步：启动中间件

```powershell
# MySQL（确保服务运行）
Get-Service MySQL* | Start-Service

# Redis（如果使用Docker）
docker run -d --name redis -p 6379:6379 redis:latest

# RabbitMQ（如果使用Docker）
docker run -d --name rabbitmq -p 5672:5672 -p 15672:15672 rabbitmq:management

# Elasticsearch（⚠️ 重要：使用17200端口）
docker run -d --name elasticsearch -p 17200:9200 -e "discovery.type=single-node" elasticsearch:8.11.0
```

### 第2步：启动后端服务（打开4个PowerShell窗口）

```powershell
# 窗口1 - Mall Service
cd E:\CFDemo\gacha-system\mall-service
mvn spring-boot:run

# 窗口2 - Game Service
cd E:\CFDemo\gacha-system\game-service
mvn spring-boot:run

# 窗口3 - Gacha Service
cd E:\CFDemo\gacha-system\gacha-service
mvn spring-boot:run

# 窗口4 - Auth Service
cd E:\CFDemo\gacha-system\auth-service
mvn spring-boot:run
```

### 第3步：启动前端

```powershell
# 窗口5 - Frontend
cd E:\CFDemo\gacha-system\game-mall
npm run dev
```

### 第4步：访问应用

浏览器打开：**http://localhost:5173**

## ❗ 常见问题

### Q1: 端口被占用怎么办？
```powershell
# 查找占用端口的进程
netstat -ano | findstr "17081"

# 结束进程
taskkill /F /PID <PID号>
```

### Q2: Maven启动很慢？
```powershell
# 首次启动需要下载依赖，请耐心等待
# 可以配置Maven国内镜像加速
```

### Q3: 数据库连接失败？
- 确认MySQL正在运行
- 确认数据库名：`gacha_system_dev`
- 确认用户名/密码：`root/123456`

### Q4: Elasticsearch连不上？
```powershell
# 确认使用了正确的端口17200
docker ps | findstr elasticsearch

# 查看ES日志
docker logs elasticsearch
```

## 💡 提示

1. **首次启动**可能需要较长时间下载依赖
2. **按顺序启动**：中间件 → 后端 → 前端
3. **查看日志**：每个服务启动时都会显示详细日志
4. **停止服务**：在每个窗口按 `Ctrl+C`
