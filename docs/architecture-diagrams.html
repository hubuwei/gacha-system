二、时序图 — 用户浏览·下单·支付

sequenceDiagram actor User as 👤 用户 participant Web as React 前端 participant Nginx as Nginx/Vite Proxy participant Auth as Auth Service :8084 participant Mall as Mall Service :8081 participant DB as MySQL participant Pay as 微信支付 User->>Web: 打开商城首页 Web->>Nginx: GET /api/games?page=0&size=20 Nginx->>Mall: 转发请求 Mall->>DB: SELECT * FROM games DB-->>Mall: 游戏列表 Mall-->>Web: {code:200, data:{content:[...]}} User->>Web: 点击登录 Web->>Nginx: POST /api/auth/login Nginx->>Auth: {username, password} Auth->>DB: 验证用户 DB-->>Auth: 用户信息 Auth-->>Web: {token, user} User->>Web: 点击游戏详情 Web->>Nginx: GET /api/games/:id Nginx->>Mall: 查询游戏详情 Mall->>DB: SELECT game + reviews DB-->>Mall: 详情数据 Mall-->>Web: {game, reviews} User->>Web: 加入购物车 → 下单 Web->>Nginx: POST /api/orders Nginx->>Mall: {gameId, quantity} Mall->>DB: INSERT orders (status=pending) DB-->>Mall: 订单创建 Mall-->>Web: {orderId, totalPrice} User->>Web: 选择支付 Web->>Nginx: POST /api/wallet/pay 或 /api/pay Nginx->>Mall: 支付请求 Mall->>Pay: 调起支付 Pay-->>Mall: 支付结果 Mall->>DB: UPDATE orders (status=paid) Mall->>DB: UPDATE wallet (扣款) Mall-->>Web: {code:200, 支付成功} Web-->>User: 🎉 购买成功！游戏已入库

三、时序图 — AI 客服对话（含打折查询）

sequenceDiagram actor User as 👤 用户 participant Web as React AI客服组件 participant Proxy as Vite Proxy participant Mall as Mall Service :8081 participant DB as MySQL participant AI as DeepSeek API User->>Web: 点击 🤖 打开客服 Web-->>User: 显示聊天面板 + 快捷提问 alt 普通问题（如"怎么退款？"） User->>Web: 输入"怎么退款？" Web->>Proxy: POST /api/ai/chat {message, history} Proxy->>Mall: 转发 Mall->>Mall: buildDiscountContext() → 无关键字 → 返回 null Mall->>AI: POST /v1/chat/completions (system + history + user) AI-->>Mall: AI 自由回复 Mall-->>Web: {code:200, data:{reply:"7天内可退款..."}} Web-->>User: 显示 AI 回复 end alt 打折查询（如"有什么打折游戏？"） User->>Web: 输入"今天有什么打折游戏？" Web->>Proxy: POST /api/ai/chat Proxy->>Mall: 转发 Mall->>Mall: buildDiscountContext() → 检测到关键字"打折" Mall->>DB: findAllDiscountedGames() DB-->>Mall: 29款打折游戏 → 取 Top10 Mall->>Mall: 格式化游戏数据注入 user message Mall->>AI: POST /v1/chat/completions (system + history + user + 折扣数据) AI-->>Mall: AI 基于真实数据生成推荐回复 Mall-->>Web: {code:200, data:{reply:"🎉 10款打折：黑神话¥149..."}} Web-->>User: 显示带真实游戏数据的回复 end alt 游戏介绍生成 User->>Web: 打开 AI 写介绍页面 Web->>Proxy: POST /api/ai/generate-game-intro {gameName, genre, style} Proxy->>Mall: 转发 Mall->>AI: 发送结构化 prompt（JSON 格式要求） AI-->>Mall: {title, summary, background, gameplay, highlights, recommendation} Mall-->>Web: {code:200, data:{...6段介绍}} Web-->>User: 分板块展示 + 一键复制 end

sequenceDiagram actor User as 👤 用户 participant Web as 前端 participant Game as Game Service :8082 participant DB as MySQL participant Redis as Redis User->>Web: 进入抽卡页面 Web->>Game: GET /api/gacha/config Game->>DB: 查询卡池配置 DB-->>Game: 卡池概率、奖池 Game-->>Web: {pools, rates} User->>Web: 点击"单抽" Web->>Game: POST /api/gacha/draw {poolId, count:1} Game->>Game: 扣减用户代币/积分 Game->>Game: 按概率随机抽取 Game->>DB: INSERT gacha_records DB-->>Game: 记录保存 Game-->>Web: {rarity, item, animation} Web-->>User: 🎆 抽卡动画 + 结果展示 User->>Web: 点击"十连抽" Web->>Game: POST /api/gacha/draw {poolId, count:10} Game->>Redis: 分布式锁(防并发) Game->>Game: 10次按概率抽取 Game->>DB: INSERT 10条 gacha_records Game->>Redis: 释放锁 Game-->>Web: [{10条结果}] Web-->>User: 🌟 十连结果展示（保底SR）
