## 🚀 快速启动

### 前置条件

1. **Java 环境**: JDK 11+
2. **MySQL 数据库**: MySQL 8.0+
3. **Maven**: 3.6+

### 启动步骤

#### 1. 初始化数据库

```bash
# 进入数据库目录
cd database

# 执行SQL脚本
mysql -u root -p < "新建文本文档.SQL"
```

#### 2. 配置数据库连接

编辑 `mall-service/src/main/resources/application.yml`:

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/gacha_system?useUnicode=true&characterEncoding=utf8&useSSL=false&serverTimezone=Asia/Shanghai
    username: root
    password: 你的密码
```

#### 3. 编译项目

```bash
cd mall-service
mvn clean package -DskipTests
```

#### 4. 启动服务

```bash
java -jar target/mall-service-1.0.0-SNAPSHOT.jar
```

或者直接运行：

```bash
mvn spring-boot:run
```

#### 5. 验证服务

服务启动后，访问：
- http://localhost:8081/api/games/all-with-tags

## 📊 核心功能说明

### 1. 钱包系统

#### 数据结构
```java
UserWallet {
    userId: 用户ID
    balance: 可用余额
    frozenBalance: 冻结余额
    totalRecharge: 累计充值
    totalConsumed: 累计消费
}
```

#### 充值流程
1. 获取或创建用户钱包
2. 记录充值前余额
3. 增加用户余额
4. 更新累计充值金额
5. 创建交易记录

#### 支付流程
1. 检查余额是否充足
2. 记录支付前余额
3. 扣除相应金额
4. 更新累计消费金额
5. 创建交易记录
6. 更新订单状态为已支付

### 2. 订单系统

#### 支持的支付方式
- `balance`: 余额支付（已实现）
- `alipay`: 支付宝（待实现）
- `wechat`: 微信支付（待实现）

#### 订单状态流转
```
pending (待支付) 
  → paid (已支付) 
  → completed (已完成)
  
pending (待支付) 
  → cancelled (已取消)
```

#### 余额支付特性
- 创建订单时立即扣款
- 余额不足自动取消订单
- 自动更新订单状态
- 自动生成交易记录

### 3. 轮播图系统

#### 特性
- 支持启用/禁用
- 支持时间范围控制
- 支持自定义排序
- 支持关联游戏或外部链接

#### 显示逻辑
只显示：
- `isActive = true`
- 当前时间在 `startTime` 和 `endTime` 之间（如果设置了的话）

## 🐛 常见问题

### 1. 编译失败
**问题**: Maven 编译报错
**解决**: 
```bash
mvn clean
mvn dependency:purge-local-repository
mvn install
```

### 2. 数据库连接失败
**问题**: Cannot connect to database
**解决**: 
- 检查 MySQL 是否启动
- 检查数据库用户名密码
- 检查数据库 `gacha_system` 是否存在

### 3. 端口被占用
**问题**: Port 8081 already in use
**解决**: 
```yaml
# 修改 application.yml
server:
  port: 8082  # 改为其他端口
```

### 4. 余额不足无法下单
**问题**: 创建订单时提示余额不足
**解决**: 
先调用充值接口为用户充值：
```bash
curl -X POST http://localhost:8081/api/wallet/recharge \
  -H "Content-Type: application/json" \
  -d '{"userId":1,"amount":100.00,"paymentMethod":"alipay"}'
```
