# 游戏商城快速启动指南

## 📋 前置条件

确保以下软件已安装并运行：

1. ✅ **JDK 11+**
2. ✅ **Maven 3.6+**
3. ✅ **MySQL 8.0+** (数据库：gacha_system)
4. ✅ **Redis 6.0+** (端口：6379)
5. ✅ **Elasticsearch 7.x** (端口：9200)
6. ✅ **RabbitMQ 3.x** (端口：5672)

## 🚀 启动步骤

### 1. 启动中间件

```bash
# 启动 Redis
redis-server

# 启动 Elasticsearch (根据系统选择对应命令)
# Windows: 运行 elasticsearch.bat
# Linux: ./elasticsearch

# 启动 RabbitMQ
rabbitmq-server
```

### 2. 配置数据库连接

编辑 `mall-service/src/main/resources/application.yml`:

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/gacha_system?useUnicode=true&characterEncoding=utf8
    username: root          # 修改为你的数据库用户名
    password: your_password # 修改为你的数据库密码
```

### 3. 配置中间件连接

编辑 `application.yml`:

```yaml
# Redis 配置
redis:
  host: localhost
  port: 6379
  password:  # 如果有密码请设置

# Elasticsearch 配置
elasticsearch:
  uris: http://localhost:9200

# RabbitMQ 配置
rabbitmq:
  host: localhost
  port: 5672
  username: guest
  password: guest
```

### 4. 配置第三方服务（可选）

```yaml
# 短信配置（阿里云）
sms:
  access-key-id: your_access_key_id
  access-key-secret: your_access_key_secret

# OSS 配置（阿里云）
oss:
  access-key-id: your_access_key_id
  access-key-secret: your_access_key_secret

# 支付配置
wechat:
  pay:
    app-id: your_app_id
    mch-id: your_mch_id
    api-v3-key: your_api_v3_key

alipay:
  app-id: your_app_id
  private-key: your_private_key
```

### 5. 编译项目

```bash
cd mall-service
mvn clean package -DskipTests
```

### 6. 启动服务

```bash
# 方式一：运行 jar 包
java -jar target/mall-service-1.0.0-SNAPSHOT.jar

# 方式二：使用 Maven 运行
mvn spring-boot:run
```

启动成功后会显示：
```
====================================
    游戏商城服务启动成功！
    访问地址：http://localhost:8081
====================================
```

## 🌐 前后端联调

### 1. 修改前端 API 地址

编辑 `game-mall/src/App.jsx` 或创建配置文件：

```javascript
const API_BASE_URL = 'http://localhost:8081/api';

// 使用示例
fetch(`${API_BASE_URL}/games`)
  .then(res => res.json())
  .then(data => console.log(data));
```

### 2. 配置 CORS（已配置）

后端已配置 CORS，允许前端跨域访问。

### 3. 前端调用示例

```javascript
// 获取游戏列表
async function getGames() {
  const response = await fetch('http://localhost:8081/api/games');
  const data = await response.json();
  console.log(data);
}

// 用户登录
async function login(username, password) {
  const response = await fetch('http://localhost:8081/api/user/login', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ username, password })
  });
  const data = await response.json();
  // 保存 token
  localStorage.setItem('token', data.data.token);
}

// 带 Token 的请求
async function getUserInfo() {
  const token = localStorage.getItem('token');
  const response = await fetch('http://localhost:8081/api/user/info', {
    headers: { 'Authorization': `Bearer ${token}` }
  });
  const data = await response.json();
  console.log(data);
}
```

## 📝 常见问题

### 1. 启动失败：端口被占用

**问题**: Port 8081 is already in use

**解决**: 
```bash
# Windows 查看占用端口的进程
netstat -ano | findstr :8081
taskkill /F /PID <PID>

# 或修改端口
server:
  port: 8082
```

### 2. 数据库连接失败

**问题**: Communications link failure

**解决**:
- 检查 MySQL 是否启动
- 检查数据库名、用户名、密码是否正确
- 检查防火墙设置

### 3. Redis 连接失败

**问题**: Cannot get Jedis connection

**解决**:
- 检查 Redis 是否启动
- 检查 Redis 配置（密码、端口）

### 4. CORS 错误

**问题**: Access to fetch at 'xxx' from origin 'null' has been blocked by CORS policy

**解决**:
- 确保后端已配置 CORS（已配置）
- 确保前端请求的 URL 正确
