## 🚀 快速开始

### 1. 启动中间件
```bash
redis-server
# 启动 Elasticsearch
# 启动 RabbitMQ
```

### 2. 导入数据库
```bash
mysql -u root -p < database/game_mall_schema.sql
```

### 3. 配置连接
编辑 `mall-service/src/main/resources/application.yml`:
- 修改数据库连接信息
- 修改 Redis 连接
- 修改其他中间件配置

### 4. 编译运行
```bash
cd mall-service
mvn clean package -DskipTests
java -jar target/mall-service-1.0.0-SNAPSHOT.jar
```

### 5. 测试接口
```bash
# 用户注册
curl -X POST http://localhost:8081/api/user/register \
  -H "Content-Type: application/json" \
  -d '{"username":"test","password":"123","phone":"13800138000"}'

# 用户登录
curl -X POST http://localhost:8081/api/user/login \
  -H "Content-Type: application/json" \
  -d '{"username":"test","password":"123"}'

# 获取游戏列表
curl http://localhost:8081/api/games
```
