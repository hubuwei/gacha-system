# 用户注册功能说明

## 功能概述

完善了用户注册功能，包含以下特性：

### ✅ 已实现功能

1. **用户名验证**
   - 长度限制：3-20 个字符
   - 格式要求：只能包含字母、数字和下划线
   - 唯一性检查：不允许重复用户名

2. **密码验证**
   - 长度限制：6-50 个字符
   - 加密存储：使用 SHA-256 哈希加密
   - 确认密码验证（RequestBody 方式）

3. **初始奖励**
   - 新用户注册赠送 1000 积分
   - 自动创建钱包账户

4. **两种接口方式**
   - `@RequestParam` - URL 参数方式
   - `@RequestBody` - JSON 请求体方式

## API 接口

### 1. 用户注册（RequestParam 方式）

**接口地址：** `POST /api/auth/register`

**请求参数：**
```
Content-Type: application/x-www-form-urlencoded

username: 用户名
password: 密码
```

**请求示例：**
```bash
curl -X POST "http://localhost:8081/api/auth/register?username=testuser&password=123456"
```

**响应示例：**
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "id": 1,
    "username": "testuser",
    "createdAt": "2026-03-29T14:00:00",
    "points": 1000,
    "balance": 0.0
  }
}
```

### 2. 用户注册（RequestBody 方式）

**接口地址：** `POST /api/auth/register/body`

**请求参数：**
```
Content-Type: application/json

{
  "username": "用户名",
  "password": "密码",
  "confirmPassword": "确认密码"
}
```

**请求示例：**
```bash
curl -X POST "http://localhost:8081/api/auth/register/body" \
  -H "Content-Type: application/json" \
  -d '{
    "username": "testuser",
    "password": "123456",
    "confirmPassword": "123456"
  }'
```

**响应示例：**
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "id": 1,
    "username": "testuser",
    "createdAt": "2026-03-29T14:00:00",
    "points": 1000,
    "balance": 0.0
  }
}
```

## 错误响应

### 常见错误码

| 错误码 | 说明 |
|--------|------|
| 400 | 请求参数错误（如密码不一致） |
| 500 | 服务器内部错误 |

### 错误信息

| 错误信息 | 说明 |
|---------|------|
| 用户名不能为空 | 用户名为空或只包含空格 |
| 用户名长度必须在 3-20 个字符之间 | 用户名长度不符合要求 |
| 用户名只能包含字母、数字和下划线 | 用户名包含非法字符 |
| 用户名已存在 | 用户名已被注册 |
| 密码不能为空 | 密码为空 |
| 密码长度必须在 6-50 个字符之间 | 密码长度不符合要求 |
| 两次输入的密码不一致 | 确认密码验证失败 |

## 使用流程

### 步骤 1：启动服务

```bash
# 进入项目根目录
cd E:\CFDemo\gacha-system

# 启动 auth-service
cd auth-service
mvn spring-boot:run
```

### 步骤 2：访问测试页面

在浏览器中打开：
- 完整版：`http://localhost:8081/register.html`
- 简化版：`http://localhost:8081/register-simple.html`

### 步骤 3：填写注册信息

1. 输入用户名（3-20 个字符，只能包含字母、数字和下划线）
2. 输入密码（6-50 个字符）
3. 确认密码（仅完整版需要）
4. 点击"立即注册"

### 步骤 4：查看注册结果

注册成功后会显示：
- 用户 ID
- 用户名
- 注册时间
- 初始积分（1000）
- 账户余额（0.0）

## 数据库表结构

### users 表

| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | BIGINT | 主键，自增 |
| username | VARCHAR(50) | 用户名，唯一 |
| password | VARCHAR(100) | 密码（SHA-256 加密） |
| createdAt | DATETIME | 创建时间 |

### wallet 表

| 字段名 | 类型 | 说明 |
|--------|------|------|
| user_id | BIGINT | 用户 ID（外键） |
| points | INT | 积分（初始 1000） |
| balance | DECIMAL | 余额（初始 0.0） |

## 安全建议

⚠️ **重要提示：**

当前实现使用 SHA-256 进行密码加密，这是基础加密方式。在生产环境中建议使用：

1. **BCrypt** - 自适应哈希函数
2. **PBKDF2** - 基于密码的密钥派生函数
3. **Argon2** - 最新的密码哈希算法

示例（BCrypt）：
```java
// 添加依赖
<dependency>
    <groupId>org.springframework.security</groupId>
    <artifactId>spring-security-crypto</artifactId>
</dependency>

// 使用 BCrypt
@Autowired
private PasswordEncoder passwordEncoder;

String encodedPassword = passwordEncoder.encode(rawPassword);
boolean matches = passwordEncoder.matches(rawPassword, encodedPassword);
```
